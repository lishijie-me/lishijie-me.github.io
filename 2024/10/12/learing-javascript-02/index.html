学习JavaScript-02：JavaScript基本概念

发表于 2024-10-12 分类于 学习
守正出奇










声明：这是我的学习总结，从网上学来的；比如 B站等等，感谢所有视频教程的作者。

JavaScript 是什么？

是一种运行在客户端（浏览器）上的编程语言，实现人机交互效果

有什么作用？

  • 网页特效（监听用户的一些行为让网页做出对应的反馈）
  • 表单验证（针对表单数据的合法性进行判断）
  • 数据交互（获取后台的数据，渲染到前端）
  • 服务端编程（nodejs）

是由什么组成的？

  • ECMAScript 语法规范
    • 规定了 JS 基础语法核心知识。如：变量、分支语句、循环语句、对象等
  • Web APIs
    • DOM 页面文档对象模型：操作文档，比如对也买你元素进行移动、大小、添加删除等操作
    • BOM 浏览器对象模型：操作浏览器，比如页面弹窗、检测窗口宽度、存储数据到浏览器等等
